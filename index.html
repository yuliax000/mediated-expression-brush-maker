How to get your brush evolve?

1. Click “Evolve”

2. Draw a circle on the canvas

3. Use sliders to increase and reduce the brush size.

4. Discover what happens.

Start creating your own brushes and artwork!

Guide

Evolve mode → G (Have a try?)

Normal mode → N (stop evolve)

Brush → B

Eraser → E

Spray tool → S

Black color palette → go back to black

Color palette → get a random color

Undo → ctrl+Z

Redo → ctrl+Y

Save Brush → save your brush to library

Export → export your artwork in png format
